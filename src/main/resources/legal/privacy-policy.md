# Política de Privacidad y Tratamiento de Datos

Versión: 2026-04-07.1

1. Sentimos Padel trata datos personales necesarios para operar la plataforma, incluyendo nombre, correo electrónico, teléfono, perfil deportivo, historial de actividad, partidos, torneos, verificaciones y métricas operativas asociadas al uso del servicio.
2. Estos datos se utilizan para:
   - autenticar usuarios
   - crear y administrar perfiles
   - gestionar partidos, torneos y clubes
   - mostrar rankings, historial y estadísticas
   - prevenir abuso, fraude o inconsistencias operativas
   - brindar soporte y comunicaciones operativas del servicio
3. La plataforma puede registrar eventos internos y actividad de uso necesarios para mantener consistencia, seguridad, auditoría operativa y mejora del producto.
4. El usuario podrá otorgar o rechazar consentimientos específicos para tracking adicional de actividad y notificaciones operativas según las opciones presentadas al registrarse o en configuraciones futuras.
5. Los datos podrán ser almacenados mientras exista una finalidad operativa, legal, de seguridad o auditoría vinculada al servicio.
6. Sentimos Padel puede actualizar esta política. Cuando corresponda, la nueva versión podrá requerir una aceptación posterior del usuario.
